## ğŸš€ Getting Started

To run these notebooks locally, you will need Python installed along with Jupyter Notebook and common data science libraries.

### Prerequisites
* Python 3.x
* Jupyter Notebook
* Pandas
* NumPy
* Scikit-learn
* Matplotlib / Seaborn

### Installation
1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/Shawnpeter578/practise_ML.git](https://github.com/Shawnpeter578/practise_ML.git)
    cd practise_ML
    ```

2.  **Install dependencies (if not already installed):**
    ```bash
    pip install notebook pandas numpy scikit-learn matplotlib seaborn
    ```

3.  **Launch Jupyter Notebook:**
    ```bash
    jupyter notebook
    ```
